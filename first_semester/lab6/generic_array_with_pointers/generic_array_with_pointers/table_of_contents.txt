Contents and short description of each file.

000_nothing.in: that's right, that file is empty.
    If you give this file as an argument to your program,
    it should create 000_nothing.res and exit.

001_array_hello_world.in: creates array, adds element to array, prints array 

002_basic_error_messages.in: test if program can handle incorrect/missing instances
    and incorrect commands.

003_put_vector_item.in : tests PutVectorItem -- index out of bounds, 
    put at beginning, put at end, put in the middle of the array, missing value.

004_get_vector_item.in : tests GetVectorItem -- index out of bounds, missing value

005_delete_vector_item.in : tests DeleteVectorItem -- delete from beginning/end,
    missing value.

006_search_vector_item.in : tests SearchVectorItem -- search for elements in the array,
    and elements not in the array.

007_sort_vector.in : tests SortVector -- sorts an unordered array.

008_merge_vectors.in : tests MergeVectos -- merges two vectors, and prints the result

009_delete_vector.in : test DeleteVector -- tests delete in various ways.
